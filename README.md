# 🎵 Spotify Music Analysis Dashboard

## Project Overview
This Power BI dashboard explores Spotify music data to analyze artist popularity, genre trends, song performance, and listening insights.

## Objectives
- Analyze music trends
- Explore artist popularity
- Visualize genre distribution
- Track song metrics

## Tools Used
- Power BI
- Excel
- Data Visualization

## Features
✅ Artist Analysis  
✅ Genre Insights  
✅ Track Popularity Metrics  
✅ Interactive Charts  

## Dataset Fields
- Artist Name
- Track Name
- Genre
- Popularity
- Streams

## Key Insights
- Popular artists identified
- Genre trends visualized
- Song popularity analysis completed
